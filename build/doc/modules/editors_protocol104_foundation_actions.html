createActions

  • createActions(lnElement: Element, lnClonedElement: Element, doElement: Element, wizard: Element, ti: string, inverted: boolean, tiInformation: TiInformation): Create[]
    • Defined in src/editors/protocol104/foundation/actions.ts:18

    Create a list of Create Actions using the parameters passed. First search for the DAI Elements that can be effected. Next create the action and add it to this list, also start the Edit Address Element wizard for all Address Elements created.

    Parameters

    • lnElement: Element

      The LN Element.

    • lnClonedElement: Element

      The Cloned LN Element, used to create new structure and determine which Create actions are needed.

    • doElement: Element

      The DO Element.

    • wizard: Element

      The Wizard to dispatch the Open Wizard event on.

    • ti: string

      The TI Value set on the new Address Elements.

    • inverted: boolean

      Indicates if the Engineer want to create inverted Address Elements, if applicable.

    • tiInformation: TiInformation

      Information about how to create the Address Elements for the passed TI.

    Returns Create[]

    A list of Create Action that will be added to the complex action.
