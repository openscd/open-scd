open-scd

OpenSCD

Open Substation Communication Designer is a bottom-up system configuration tool for projects described using IEC 61850-6 Edition 2 or greater.

Installation

 
git clone git@grumpy.omicron.at:main/open-scd.git
cd open-scd
npm install
npm start

Scripts
  • start runs OpenSCD for development, reloading on file changes
  • test runs the test suite with Karma
  • lint runs the linter
  • doc builds markdown documentation in the doc directory
